# vimrc - next generation

My next generation vimrc.

## Requirements

* `rg` (ripgrep)
* `fzf`

## Installation

First, initialize the submodules:

```
$ ./update.sh
```

### Vim

```
$ ln -s $PWD/ ~/.vim
```

### Neovim

```
$ mkdir -p ~/.config/nvim
$ ln -s $PWD/ ~/.config/nvim
```

## Updating

```
$ ./update.sh
```
